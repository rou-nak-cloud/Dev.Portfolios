# 📂 Dev.Portfolios

Welcome to my portfolio laboratory. This repository is a centralized **"Market of Portfolios"**—a collection of various personal portfolio designs, experiments in UI/UX, and creative frontend implementations.

## 🚀 Concept
Instead of having multiple scattered repositories, I maintain all my portfolio versions here. This allows me to track my growth as a developer and experiment with different tech stacks (React, Tailwind CSS, GSAP, Lenis) in one place.

## 🛠️ Tech Hub
Across these projects, I focus on:
* **Modern Styling:** Tailwind CSS v4, Framer Motion, and Magic UI components.
* **Smooth Motion:** Lenis for smooth scrolling and GSAP for scroll-triggered interactions.
* **Performance:** Optimized asset loading and clean React architecture.

## 📖 How to Navigate
1. Open a specific folder (e.g., `/Portfolio_v3.1`).
2. Follow the `README.md` inside that folder for specific setup instructions.
3. Run `npm install` and `npm run dev` to see the project in action.
